# library_system

Авторы: Балдин Р.A, Сальников Влад, Гусев И.Ю, Иванов К.Н

Электронная система учета книг в библиотеке, база данных.


Проблема, которую решает наш проект: 
	•	Привычный способ учета книг требует больших затрат человеческого ресурса, канцелярских принадлежностей, бумаги и т.д. 
	•	Существует вероятность ошибки из-за человеческого фактора. 


Цель проекта:
	•	Цифровизировать базу данных о книгах и учениках.
	•	Создать удобный и понятный обывателю интерфейс.
	•	Сделать систему которую легко внедрить в учебные организации.

Проект состоит из:
	•	Графический интерфейс, через который осуществляется доступ к каждой книге.
	•	Для каждой книги: последние изменения произошедшие с ней, информация о том «взята» ли книга, если да, то кем.
	•	Для каждого ученика: данные о том какие книги брал, какие вернул и т.п.
	•	Физический модуль на базе Arduino, который позволяет идентифицировать пользователя по пропуску.
