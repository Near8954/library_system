# library_system

Авторы: Балдин Р.A, Гусев И.Ю, Иванов К.Н

Электронная система учета книг в библиотеке, база данных.


Проблема, которую решает наш проект: 
	•	Привычный способ учета книг требует больших затрат человеческого ресурса, канцелярских принадлежностей, бумаги и т.д. 
	•	Существует вероятность ошибки из-за человеческого фактора. 


Цель проекта:
	•	Цифровизировать базу данных о книгах и учениках.
	•	Создать удобный и понятный обывателю интерфейс.
